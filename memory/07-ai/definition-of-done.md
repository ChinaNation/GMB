# GMB Definition Of Done

一个任务只有同时满足以下条件，才算真正完成：

## 1. 功能完成

- 功能达到任务目标
- 没有明显行为回归
- 不存在为了迁就历史状态而保留的旧流程、旧格式、旧数据、旧命名、旧目录、旧接口分支或旧 UI 文案

## 2. 代码完成

- 关键逻辑已补中文注释
- 没有临时调试残留
- 没有待办、修复、临时占位、调试打印等残留标记
- 旧代码、旧注释、旧配置、旧测试和废弃分支已按目标态彻底清理

## 3. 文档完成

- 已更新 `memory/` 或对应技术文档
- 如涉及边界变化，已更新架构文档或 ADR
- 旧文档、旧任务描述和旧口径残留已删除或改为目标态描述,不得保留兼容旧流程的说明

## 4. 验证完成

- 已运行必要测试
- PR 门禁通过
- Claude Review 主要问题已处理
- 涉及 API、数据库、登录、权限、扫码或页面展示时，已完成真实运行态验收；只跑编译、类型检查、单元测试或前端 build 不算完成

## 5. 沟通完成

- 如需求或逻辑存在不确定性，已与负责人确认
