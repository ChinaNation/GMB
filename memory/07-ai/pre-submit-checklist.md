# GMB 提交前收口清单

## 代码收口

- 关键逻辑已补中文注释
- 没有临时调试代码、无用输出和废弃分支
- 没有保留旧流程、旧格式、旧命名、旧目录、旧接口分支、过渡兼容或影子旧流程

## 文档收口

- `memory/` 或模块技术文档已同步更新
- 如有边界、结构或契约变化，已补 ADR 或模块说明
- 旧注释、旧文档、旧任务描述和旧 UI 文案残留已同步删除或改为目标态

## 任务收口

- 任务卡中的待确认问题已处理或注明原因
- 模块执行清单已对照过
- 模块级完成标准已对照过
- 涉及 API、数据库、登录、权限、扫码或页面展示时，已记录真实服务、真实数据库、真实 HTTP 接口或真实页面的验收结果

## 残留清理

- 无用文件、无用依赖、无用配置已清理
- 演示数据、临时任务卡和试验文件没有残留
- 旧数据和半成品数据已按目标态清理；除非用户明确要求兼容，否则不得把旧数据补字段后继续保留
