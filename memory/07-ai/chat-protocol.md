# GMB 聊天协议

## 1. 对你的输入要求

你在 Codex 聊天窗口里直接输入：

- 中文任务需求

## 2. AI 的默认首轮响应

AI 在第一轮不直接写代码，而是必须先做需求分析。

第一轮正式回复必须：

- 以 `需求分析` 开头
- 先概括任务需求
- 输出建议模块、影响范围、主要风险点、是否需要先沟通、建议下一步
- 不直接进入实现
- 不跳过任务卡直接开工

标准分析格式见：

- `memory/07-ai/requirement-analysis-template.md`

## 3. 多线程下的聊天协议

即使你在 Codex 里开多个线程，也要遵守同一套协议：

- 第一轮先分析
- 关键结论写回 `memory/` 或任务卡
- 不依赖某个线程私有的聊天历史当长期记忆
