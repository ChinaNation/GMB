# GMB 模块文档目录

`memory/05-modules/` 是 GMB 全仓库的模块级技术文档真源目录。

这里统一存放：

- `citizenchain` 模块文档
- `sfid` 模块文档
- `cpms` 模块文档
- `wuminapp` 模块文档

## 规则

- 产品目录中的正式模块文档统一迁入这里。
- 产品目录默认只保留代码、配置、测试、构建脚本和资源文件。
- 模块文档必须描述需求、边界、实现、关键数据结构与回写要求。
- 改模块代码后，必须同步回写这里的对应模块文档。
