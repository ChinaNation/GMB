## Phase 2 目标（跨产品）

### 2A: 协议新增 spec_version（Q2+Q12）

**问题**：链升级插入新 pallet 后 index 偏移，冷钱包解码器过时。
**方案**：sign_request 新增可选字段 `spec_version`，冷钱包校验版本是否已知。

改动范围：
- `wumin/lib/signer/qr_signer.dart` — QrSignRequest 新增 specVersion
- `wuminapp/lib/signer/qr_signer.dart` — 同步
- `wumin/lib/signer/pallet_registry.dart` — 增加 supportedSpecVersions
- `wumin/lib/signer/payload_decoder.dart` — decode() 增加 specVersion 参数
- `wumin/lib/signer/offline_sign_service.dart` — 传递 specVersion
- `wuminapp/lib/trade/onchain/onchain_trade_page.dart` — buildRequest 传入 specVersion
- `wuminapp/lib/governance/transfer_proposal_page.dart` — 同上
- `wuminapp/lib/governance/transfer_proposal_detail_page.dart` — 同上
- `wuminapp/lib/governance/runtime_upgrade_page.dart` — 同上
- `wuminapp/lib/governance/runtime_upgrade_detail_page.dart` — 同上

### 2B: wuminapp BIP39 自动提示（Q4）

- `wuminapp/lib/ui/widgets/bip39_input.dart` — 复制组件
- `wuminapp/lib/wallet/ui/wallet_page.dart` — ImportWalletPage 使用 Bip39InputField

### 2C: wuminapp 截屏保护（Q10）

- `wuminapp/lib/util/screenshot_guard.dart` — 复制工具类
- `wuminapp/android/.../MainActivity.kt` — FLAG_SECURE MethodChannel
- `wuminapp/ios/Runner/AppDelegate.swift` — blur 保护
- `wuminapp/lib/wallet/ui/wallet_page.dart` — 创建钱包助记词弹窗添加保护

## 不兼容说明

- `spec_version` 为可选字段，向后兼容；旧版 wuminapp 不发送时冷钱包按 decodeFailed 处理
- BIP39 和截屏保护为 UI 层改动，无协议影响
