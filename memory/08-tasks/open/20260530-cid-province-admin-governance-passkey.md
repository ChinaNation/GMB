# OnChina 联邦注册局机构管理员治理冷钱包扫码签名任务卡

## 任务目标

修复联邦注册局机构管理员治理安全模型：联邦注册局机构管理员只负责管理员治理，不再持有或使用云端业务签名私钥；当前目标模型已收敛为登录态 + `QR_V1` 冷钱包 sr25519 扫码签名。

## 固定约束

- 全仓库只允许 `QR_V1` 一个二维码协议。
- 不新增 `ONCHINA_ADMIN_ACTION_V1` 或其它协议。
- 不做旧 signer 兼容，不迁移旧 seed，不读取旧路径。
- 不使用的代码、注释、文档必须清理。

## 范围

- 查看市注册局机构管理员列表：保留登录态访问。
- 新增、删除市注册局机构管理员：必须登录态 + 当前联邦注册局机构管理员公民钱包签名。
- 编辑市注册局机构管理员姓名：只需要登录态，后端仍校验角色和省域。
- 联邦注册局机构管理员不得本地新增或删除,只能通过 `REPLACE_FEDERAL_REGISTRY` 同省更换。
- 编辑联邦注册局机构管理员姓名：只需要登录态，后端仍校验角色和省域。
- 清理联邦注册局机构管理员云端代签逻辑。
