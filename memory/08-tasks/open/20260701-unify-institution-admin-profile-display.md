# 任务卡：统一机构管理员岗位任职展示

## 当前状态

- 状态：已被 `20260628-institution-admin-field-model-onchain.md` 替代并完成迁移

## 替代说明

原任务把姓名、公民 CID、岗位、任期和来源内嵌进机构管理员集合。该模型已经废弃。

当前统一展示契约为：

- admins 提供 `admin_account + family_name + given_name` 管理员人员集合；授权只比较钱包账户。
- entity 提供机构岗位和管理员任职唯一真源。
- 前端按一个管理员钱包聚合全部有效任职，展示钱包、余额、岗位、任期、来源和来源引用。
- 普通公民姓名和 CID 不从管理员钱包推导或展示。
- Node、OnChina、CitizenApp 均已使用岗位任职卡片；旧资料卡和机构管理员集合编辑入口已删除。
- 个人多签管理员保持独立模型，不使用机构岗位任职卡片。
