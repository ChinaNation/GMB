# SFID 机构列表操作记录默认账户改造任务卡

## 任务目标

按用户确认要求，统一修正 SFID 机构列表展示、首页公民身份列表搜索入口、机构完整操作记录、机构简称、机构类型标签，以及国储会/省储行默认账户显示与数据。

## 范围

- 前端：公权机构正常状态标签改为绿色，市公安局列名改为“市公安局名称”。
- 前端：所有机构列表删除“创建用户”列，不保留旧列口径。
- 前端：首页公民身份列表不保留独立查询按钮，搜索图标内置到搜索框右侧，“新增公民”按钮位于搜索框右侧。
- 前端：机构操作记录展示机构完整操作记录，覆盖机构创建、详情编辑、账户创建/删除、资料上传/下载/删除、CPMS 安装码吊销等。
- 前端/后端：修正公权机构简称生成与既有数据，避免简称显示全称。
- 前端/后端：机构类型按主体属性显示为“公民储备委员会 / 国储会”等目标口径。
- 后端/数据库：省公民储备银行补充“永久质押”默认账户；国储会补充“安全基金”“两和基金”默认账户；名称按系统实际保留名称处理。
- 文档：更新 SFID 前端、后端、审计和主体规则文档。
- 残留：清理旧列、旧标签、旧简称和旧账户数量口径。

## 不做

- 不新增独立 `sfid/backend/src/` 源码壳。
- 不新增独立 `frontend/api/` 业务目录。
- 不引入旧口径兼容分支；按目标状态一次性改造。
- 不修改投票引擎职责边界。

## 已确认

- 操作记录包含全部机构相关行为。
- 默认账户名称以系统实际保留名称为准。
