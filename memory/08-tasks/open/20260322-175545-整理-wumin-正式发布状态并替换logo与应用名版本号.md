# 整理 wumin 正式发布状态并替换 logo 与应用名版本号

## 任务背景

用户希望将 `wumin` 冷钱包整理为正式发布状态，包括：

- 使用仓库 `docs` 中的 logo 作为安卓安装包图标
- 调整应用名
- 调整版本号

## 当前状态

- `wumin` 已具备正式签名构建能力
- 当前安卓应用名仍为 `公民钱包`
- 当前版本号仍为 `0.1.0+1`
- 当前 launcher icon 仍是默认资源

## 风险点

- 若 logo 资源路径判断错误，可能替换到错误素材
- 若只改前景图不改 adaptive icon 背景，桌面图标可能显示异常
- 若版本号未同步递增，后续安装覆盖可能失败

## 执行计划

1. 定位仓库内可用 logo 资源
2. 替换 `wumin` Android launcher icon
3. 调整应用名与版本号到正式发布状态
4. 重新构建正式签名 APK 验证结果

## 实际处理

- 使用 `/Users/rhett/GMB/docs/logo.png` 生成并替换了 `wumin` Android 全部 launcher icon 资源
- 同步替换了 `wumin` iOS `AppIcon.appiconset`，保持两端视觉一致
