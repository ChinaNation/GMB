# 任务卡：管理员模块跨端边界整理

## 当前状态

- 状态：原机构管理员直接变更方案已被 `20260628-institution-admin-field-model-onchain.md` 替代

## 当前边界

- Node 只读取机构 admins 钱包集合、entity 岗位任职，并提供本机管理员激活；不提供机构管理员集合编辑、差异或提交页面。
- CitizenApp 的管理员集合变更只保留 `PMUL` 个人多签；公权、私权、固定治理和非法人机构均不得进入个人多签变更流程。
- 机构岗位或管理员任职变化由具体注册、普选、互选、提名任免等业务形成治理结果，再由 entity 原子更新任职并派生 admins。
- 投票流程统一由投票引擎承担，业务模块不得自行实现计票和终态判定。
- CitizenWallet 只解码和签署当前仍存在的个人多签管理员变更协议；旧公权/私权机构动作码和解码分支已删除。
- 个人多签与机构管理员完全独立，不共享岗位、任职或管理员变更协议。
