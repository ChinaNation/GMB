## 目标

整理钱包 UI 中散落的扫码入口，统一到热钱包详情页右上角。

## 变更清单

1. 删除我的钱包列表中冷/热钱包卡片左下角的扫码图标
2. 删除热钱包详情页中的"离线签名"按钮和说明文字
3. 在热钱包详情页 AppBar 右上角新增扫码图标
   - 点击打开扫码页面
   - 扫到登录码 → 走登录流程
   - 扫到签名请求 → 走离线签名流程
   - 扫到其他类型 → 提示"非签名或登录码"
