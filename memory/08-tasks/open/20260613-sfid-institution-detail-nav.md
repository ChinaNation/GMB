# SFID 机构详情连接式导航改造任务卡

## 任务目标

将 SFID 各类机构详情页统一改为参考 BrixUI Tabs 的“连接式左侧导航 + 右侧内容区”布局，并把市注册局联邦管理员视角的城市入口改为参考市公安局的表格列表。

## 范围

- 前端：统一公权机构、公安局、注册局、教育机构、私权机构详情页导航布局。
- 前端：拆出可复用的机构操作记录组件，所有机构详情页都展示操作记录。
- 前端：无管理员数据的机构详情页不展示“管理员列表”tab。
- 前端：市注册局城市入口改为表格，最后一列展示“管理员数”，格式为 `x / 30`。
- 前端：机构详情账户列表补充序号列，联邦注册局账户新建按钮按管理员角色显示。
- 后端：公权机构只读列表支持 `org_code` 精确过滤，供市注册局表格一次性读取完整身份ID。
- 文档：更新 SFID 前端布局文档。
- 残留：清理被连接式详情布局替换的注释和市注册局城市入口口径。

## 不做

- 不新增独立后端接口路径；只在既有公权机构只读列表上增加过滤参数。
- 不改变现有账户、资料库、管理员新增删除、Passkey 和公民钱包签名流程。
- 详情页仅保留连接式导航布局，不设置并行实现。
