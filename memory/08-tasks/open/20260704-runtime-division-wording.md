# 任务卡：runtime 行政区层级中文描述统一

## 任务需求

将 `citizenchain/runtime/` 内中文注释、文档注释、测试说明和 panic 文案中的行政区层级描述统一为目标口径：

- `国家级` -> `国家`
- `省级` -> `省行政区`
- `市级` -> `市行政区`
- `镇级` -> `镇行政区`
- `省/国家级` -> `省行政区/国家`
- `省/国级` -> `省行政区/国家`

本任务只改中文描述，不修改链上逻辑、类型、编码、storage key、交易载荷或创世数据。
