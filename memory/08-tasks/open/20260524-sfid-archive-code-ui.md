# SFID 扫描档案码弹窗 UI 调整

## 任务需求

SFID 公民绑定弹窗中，统一可见口径为“档案码”：

- 将弹窗标题改为“扫描档案码”，并居中显示。
- 删除新账户绑定档案的旧模式说明。
- 删除新账户绑定档案的旧步骤标题。
- 将扫码框提示改为“点击扫描档案码”。
- 本任务只处理 UI 展示和组件内残留，不改后端协议、不改 CPMS 档案码载荷。

## 建议模块

SFID 前端公民绑定弹窗。

## 影响范围

- `sfid/frontend/citizens/BindModal.tsx`
- `sfid/frontend/citizens/api.ts`
- `memory/05-modules/sfid/frontend/FRONTEND_LAYOUT.md`

## 主要风险点

- 前端组件必须清理旧档案码可见文案和内部说明残留。
- 后端当前仍接收既有绑定字段，本任务不扩大为协议改造。

## 验收标准

- 弹窗标题为“扫描档案码”且居中。
- 新账户绑定档案时不展示模式说明和旧步骤标题。
- 扫码框提示为“点击扫描档案码”。
- SFID 前端公民绑定组件中不再残留旧档案码文案或命名。
- 前端构建检查通过。
