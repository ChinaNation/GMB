## 目标

将冷钱包离线签名功能从 wuminapp 中完全剥离，建立独立的 wumin app 作为"软件形态的硬件钱包"，实现冷热钱包的物理隔离。

## 背景

- 冷钱包私钥不应出现在联网设备上
- wumin app 是一个最小化的离线签名设备，无网络权限
- 通过 QR 二维码（V2 协议）与 wuminapp 交互

## 实现内容

### 1. 新建 wumin Flutter 项目 `/GMB/wumin/`

- **pubspec.yaml**: 最小依赖集，不含 http/websocket/polkadart 等网络库
- **Isar schema**: 仅保留 WalletProfileEntity / WalletSettingsEntity / AppKvEntity
- **WalletManager**: 仅支持热钱包 create/import/delete/rename/sign，无冷钱包方法
- **签名核心**: 迁移 qr_signer.dart / offline_sign_service.dart / payload_decoder.dart
- **UI 页面**:
  - `main.dart` → MaterialApp 入口
  - `home_page.dart` → 钱包列表 + 创建/导入入口 + 扫码签名
  - `create_wallet_page.dart` → 创建钱包 + 展示助记词
  - `import_wallet_page.dart` → 通过助记词导入钱包
  - `offline_sign_page.dart` → 扫码签名页（扫描→验证→签名→展示回执二维码）

### 2. 清理 wuminapp

- 删除 `lib/signer/offline_sign_service.dart`
- 删除 `lib/signer/payload_decoder.dart`
- 删除 `lib/qr/pages/qr_offline_sign_page.dart`
- 删除 `test/signer/offline_sign_service_test.dart`
- 删除 `test/signer/payload_decoder_test.dart`
- 移除 `signer.dart` barrel 中的 `offline_sign_service` export
- `wallet_page.dart`: 扫到 sign 类型时提示"请使用 Wumin 签名设备"

### 3. 迁移测试

- `test/signer/offline_sign_service_test.dart` → wumin（移除冷钱包测试用例）
- `test/signer/payload_decoder_test.dart` → wumin（覆盖 transfer/vote/joint_vote/citizen_vote）
