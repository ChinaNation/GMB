# CitizenApp 私密聊天

- 状态：已由 `20260711-chat-square-step1.md` 彻底改造
- 模块：`citizenapp`

## 最终架构

- 钱包账户是聊天账户，钱包主私钥只用于短期会话与设备绑定授权，不签署每条聊天消息。
- OpenMLS 提供端到端加密；消息、会话、联系人关系和附件只保存在通信双方设备。
- Cloudflare 只保存设备公钥、推送 Token、一次性 KeyPackage 和防重放摘要。
- 文本密文通过休眠 WebSocket 瞬时转发，附件通过 WebRTC DataChannel 设备间传输。
- 接收方暂不可达时发送方保留本地出站队列；通用推送唤醒接收方后，双方设备自动建立连接并继续投递，不要求两人同时打开页面。
- 用户注销后，设备本地聊天数据和 Cloudflare 必需活动数据立即硬删除。

## 永久边界

- 不使用区块链 runtime、链节点或链上交易传输聊天。
- 不在 Cloudflare 持久化消息、附件、会话或联系人关系。
- 不恢复任何旧传输、旧字段、旧表或兼容流程。
