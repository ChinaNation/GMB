# SFID 省管理员治理 Passkey 与冷钱包签名第 1 步任务卡

## 任务目标

修复省管理员治理安全模型：省管理员只负责管理员治理，不再持有或使用云端业务签名私钥；省管理员写操作必须通过登录态、Passkey、`WUMIN_QR_V1` 冷钱包 sr25519 挑战签名。

## 固定约束

- SFID 域名固定为 `sfid.crcfrcn.com`。
- 全仓库只允许 `WUMIN_QR_V1` 一个二维码协议。
- 不新增 `SFID_ADMIN_ACTION_V1` 或其它协议。
- 不做旧 signer 兼容，不迁移旧 seed，不读取旧路径。
- 不使用的代码、注释、文档必须清理。

## 范围

- 查看市管理员列表：保留登录态访问。
- 新增、编辑、删除市管理员：必须登录态 + Passkey + 当前省级管理员冷钱包签名。
- 新增、编辑省级管理员：必须登录态 + Passkey + 当前省级管理员冷钱包签名。
- 删除新增省级管理员：必须由本省初始省级管理员通过 Passkey + 冷钱包签名提交。
- 删除省管理员云端代签逻辑。
