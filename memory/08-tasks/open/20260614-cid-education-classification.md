# CID 教育机构分类与法人教育分支

## 任务需求

调整 CID 教育机构分类与展示:

- 教育机构 tab 统一管理国家公民教育委员会、市公民教育委员会、法人教育机构和既有 `F + JY` 非法人教育机构。
- 已生成的市公民教育委员会进入所属市教育机构 tab 后直接显示,不要求输入名称或身份ID;国家公民教育委员会不跨市直显。
- 法人教育机构和 `F + JY` 非法人教育机构仍按名称或身份ID搜索后显示。
- 新增学校时只允许选择初学、小学、中学、大学。
- 学校内部部门不作为独立 CID 机构主体,不生成身份ID,不进机构列表和详情 tab。
- 法定代表人选择改为按目标机构类型决定范围:普通私法人/挂靠私法人的机构可选全国正常公民;公权机构、公法人教育机构和挂靠公法人的非法人机构必须按机构所属省/市选择本地正常公民;国家/部/联邦级公权机构可选全国正常公民。

## 硬边界

- 不修改 CID 号生成逻辑。
- 不修改 `G/S/F + JY` 编码规则。
- 不新增第二套身份ID字段或身份ID规则。
- 所有机构主身份字段仍然只允许使用 `cid_number`。
- `F + JY` 必须保留,继续作为非法人教育机构/法人教育机构分支的 CID 生成依据。
- 学校内部部门不得作为新增弹窗手动选项。
- 禁止新增第二套法定代表人身份ID字段;机构仍只保存 `legal_rep_cid_number`,并指向公民唯一 `cid_number`。
