# SFID 管理员模块目录统一任务卡

## 任务目标

将 SFID 管理员相关前后端模块统一收口到 `admins` 目录，不保留旧省级/市注册局机构管理员双目录。

## 范围

- 后端：把联邦注册局机构管理员、市注册局机构管理员账号维护、管理员列表和管理员安全写入口统一到 `sfid/backend/admins/`。
- 前端：把联邦注册局机构管理员页面、市注册局机构管理员页面和管理员 API 统一到 `sfid/frontend/admins/`。
- 文档：更新 SFID 技术文档中的管理员模块目录口径。
- 残留：清理旧目录引用、旧注释和旧默认存储路径。

## 不做

- 不做旧目录兼容。
- 不做旧管理员 signer / seed / cache 兼容读取。
- 不改变现有外部 API 路径语义。
