# 升级冷钱包签名协议为 WUMIN_SIGN_V1.0.0，解决盲签问题

## 状态：进行中

## 背景

当前 `WUMIN_SIGN_V1.0.0` 协议存在盲签漏洞：离线设备对 `payload_hex` 原样签名，不解码、不展示交易内容。用户在离线设备上无法确认实际签名的交易是什么。

## 目标

- 协议直接升级为 `WUMIN_SIGN_V1.0.0`，不兼容 V1
- 在线设备构建 sign_request 时附带人可读 `display` 摘要
- 离线设备端独立解码 payload，与 display 交叉比对
- 比对一致才允许签名，不一致则阻止
- sign_response 回传 `scope` 和 `payload_hash`
