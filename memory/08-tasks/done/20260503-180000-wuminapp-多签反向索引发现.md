# wuminapp 多签反向索引发现 + 合并入口 + 删除扫描加入

## 背景

wuminapp 当前 "我的多签" 数据源是本地 Isar(`PersonalDuoqianEntity` / `DuoqianInstitutionEntity`),只能看到本机创建/手动添加的多签。任意本钱包账户作为 admin 参与的多签若由他人创建,本机看不到 — 必须依赖创建者出 QR、本机扫描加入(`kind=user_duoqian` 协议)。

链端的 `AdminsChange::Institutions` 已经是个人/SFID/内置三类多签的统一存储,value 含 admins 列表 + kind 字段。smoldot 完全支持 `state_getKeysPaged`(verified [smoldot-pow/lib/src/json_rpc/methods.rs:432](wuminapp/smoldot-pow/lib/src/json_rpc/methods.rs:432))。客户端遍历该 storage 即可实现反向索引发现,**0 链端改动**。

发现机制到位后,扫码加入冗余,连同 `kind=user_duoqian` QR 协议一并下线。

## 边界(死规则)

- 链端 0 改动(spec_version 不变 / chainspec / runtime / pallet / metadata 全不动)
- `discoveredViaAdmin=false` 的 entity(本机创建/手动添加)永不被反向校验删除
- 不动机构多签创建流程
- 不动详情页主体逻辑(只删"分享 QR"按钮)

## 范围概览

### 反向索引(req 3 核心)
- `DuoqianDiscoveryService` 全扫 `AdminsChange::Institutions` + 过滤 `kind ∈ {1=Sfid, 2=Personal}` + `admins ∩ myPubkeys ≠ ∅`
- 命中后按 kind 反查 `PersonalDuoqianInfo` 或翻页 `SfidRegisteredAccount` 拿全部 account
- upsert Isar(标记 `discoveredViaAdmin=true` + `matchedAdminPubkeys`)
- 反向校验:Isar 中 `discoveredViaAdmin=true` 但本次未命中的 entity 全部删除
- 30 分钟节流 + 后台静默扫描

### UI 合并(单"多签交易"入口)
- `transaction_tab_page` 双入口 → 单"多签交易"
- `DuoqianAccountListPage` 删 `accountType` 参数 + 合并展示 + icon 区分 + UI hint(我创建 / 我作为 N 位管理员之一)
- 创建按钮"+"sheet:创建个人多签 / 创建机构多签(2 个选项)
- 顶部 banner:扫描中 X / Y · 已发现 M

### 删除"扫描加入多签"全部实现
- `kind=user_duoqian` QR 协议链路全删(envelope / qr_protocols / user_duoqian_body / duoqian_qr_sheet)
- `DuoqianAccountInfoPage` 删"分享 QR"按钮 + 调用
- `DuoqianAccountListPage` 删扫码入口 + handler
- 壳子文件 `PersonalDuoqianListPage` / `InstitutionDuoqianListPage` / `DuoqianAccountType` 全删

### 设计实现要点

1. **链端 0 改动**:利用现有 `AdminsChange::Institutions`(三类多签统一存储)+ smoldot 标准 `state_getKeysPaged`(verified [methods.rs:432](wuminapp/smoldot-pow/lib/src/json_rpc/methods.rs:432))。
2. **个人多签 institution_id = `personal_address(32B) || zeros(16B)`**,对齐 [duoqian-manage/common.rs:19](citizenchain/runtime/transaction/duoqian-manage/src/common.rs:19)。
3. **SFID 机构 institution_id = `sfid_number_utf8 || zeros padded 48B`**,对齐 [duoqian-manage/common.rs:64](citizenchain/runtime/transaction/duoqian-manage/src/common.rs:64);需翻页 `SfidRegisteredAccount` 拿全部 (sfid_number, account_name) → duoqian_account 关联。
4. **30 分钟节流 + SharedPreferences 持久化**,下拉刷新强制 force=true。
5. **反向校验**:Isar 中 `discoveredViaAdmin=true` 但本次未命中的 entity 全部删除;`discoveredViaAdmin=false`(本机创建/手动添加)永不被自动清理。
6. **三类删除场景统一处理**:多签关闭 / 用户钱包删除某账户 / admin 被换出(目前链端仅 BuiltinInstitution 支持换 admin,但反向校验逻辑前瞻兼容)。
