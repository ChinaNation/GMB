# 任务卡：身份ID系统行政区数据统一为「省 / 市 / 镇」三级

## 目标

把 CID 行政区真源 `china.sqlite` 里镇一级五花八门的命名（街道/乡/开发区/林场/农场/苏木/管委会…）统一改写为「X镇」，对外只承认 **省 / 市 / 镇** 三级（镇下挂村/路，村级本次不动）。

边界（已查实）：
- **市级已 100% 统一**（3,185 个全「X市」）→ 不碰。
- 真源单一文件 `citizencode/backend/china/china.sqlite`（git-LFS，运行时由 `china/store.rs` 加载，无 Rust 硬编码镇表、无生成脚本）。全仓**仅此一份**，CPMS 后端经编译期相对路径引用同一文件（`citizenpassport/backend/initialize/mod.rs:797`），无第二副本。
- **功能安全**：全仓无任何代码按「街道/镇/乡」后缀做逻辑分支；CID 号只编码省+市（R5），镇名仅 `subjects.town` 展示字段 → 改名不影响已签发号码、不动链、不重新创世。`town_code`=(province_code,city_code,code) 才是唯一键，同名不破坏数据。
