# 任务卡：SFID 机构法定代表人与机构详情清理

## 目标

按目标状态调整 SFID 机构展示与机构资料：

- 机构详情页把“机构 SFID”改为“身份ID”，并去掉身份ID外层代码框。
- 机构详情页删除“A3 类型”和“链上状态”两行；机构本身不再展示链上状态，链上状态只属于账户。
- 公权机构、公安局、私权机构三个列表增加连续“序号”列。
- 机构新增法定代表人资料：姓名、身份ID、证件照。
- 法定代表人身份ID只能从正常状态公民中选择；选择接口只返回 `sfid_number`。
- 新增机构必须填写法定代表人姓名、身份ID、证件照；已初始化机构可暂为空，但后续编辑时必须补齐。

## 预计修改目录

- `sfid/backend/core/`：调整数据库目标结构和索引，增加机构法定代表人字段，移除机构链上状态业务字段残留。
- `sfid/backend/subjects/`：调整机构 DTO、详情/列表/编辑接口、法定代表人校验和照片上传。
- `sfid/backend/private/`：调整私权机构新增与编辑校验，确保新建机构必须填写法定代表人资料。
- `sfid/backend/gov/`：调整公权机构初始化与列表详情输出，保留初始化机构法定代表人可空。
- `sfid/frontend/subjects/`：调整机构 API 类型，删除机构链上状态类型，增加法定代表人字段。
- `sfid/frontend/gov/`：调整公权机构/公安局列表序号与详情页字段展示。
- `sfid/frontend/private/`：调整私权机构列表序号、详情页字段展示和编辑表单必填校验。
- `sfid/frontend/core/institution/`：调整机构创建表单，新增法定代表人填写、选择和证件照上传。
- `memory/01-architecture/sfid/`：更新 SFID 架构文档，记录机构法定代表人与链上状态边界。
- `memory/08-tasks/`：任务状态流转与验收记录。
