# SFID 省管理员备用管理员扫码新增与卡片布局整改

## 任务来源

用户要求整改注册局-省级管理员页面:

- 备用管理员为空时,对应卡片显示新增按钮。
- 新增备用管理员必须扫码填入账户。
- 三个管理员卡片竖向排列。
- 删除页面顶部槽位提示。
- 管理员卡片显示并设置管理员姓名。
- 卡片字段“状态”改为“签名密钥”。

## 本次目标

- 前端完成三槽卡片竖向展示和扫码新增备用管理员弹窗。
- 后端补本地备用管理员保存接口,让刷新后仍能显示备用管理员姓名和账户。
- 保持签名密钥由管理员本人登录后自动生成或手动生成/更换。
- 账户只允许扫码填入。
