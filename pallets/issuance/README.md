# 1.省储行创立发行质押利息/shengbank-stake-interest
## 1.1.功能需求
* 初始省储行节点，每个省储行发行该省总人口数x100的数字公民币，各省人口以中共第7次人口普查数据为基准（并做省份调整），共计发行144,349,737,800元；
* 各省储行创立发行的数字公民币永久质押于该省储行质押地址（keyless_address），该地址为无私钥地址，永久质押；
* 各省储行质押的数字公民币，由区块链发行质押利息,质押利息存入各省储行地址（pallet_address），利息归各省储行所有，用于省储行运营和资助民运人士；
* 质押利息初始年利率为1%，并以线性衰减的方式每年减少0.01%，100年后停止计算利息，共产生利息72,896,617,589.00元公民币；  
* 省储行keyless质押地址的生成规则为：GMB_SHENGBANK_STAKE_ADDRESS_省代码_省名称，详见tools/keyless。

# 2.全节点发行/fullnode-pow-reward
## 2.1.功能需求
* 运行全节点的每铸造一个新区块，系统发行9999.00元数字公民币用于奖励该节点，全节点铸块发行为第1个至第9,999,999个区块，共计发行量为99,989,990,001.00元数字公民币；
* 当区块高度超过9,999,999个区块后（第10,000,000个起，含本数），即永久停止全节点发行，全节点再铸造新块的将不再获得铸块奖励。
   |  单个区块发行量 |   可发行的区块总量   |       总发行量          |     简述      |
   |:-------------:|:-----------------:|:----------------------:|:------------:|
   |   9999.00元   |    9,999,999个     |  99,989,990,001.00元   |  999.89亿元   |
* 全节点发行模块仅负责发行，不参与铸块，全节点通过PoW工作量证明获得铸块权后，由全节点发行模块发行公民币予以奖励；

# 3.公民轻节点发行/citizen-lightnode-issuance
## 3.1.功能需求
* 使用联邦公民身份识别码完成公民轻节点认证的，将获得认证奖励，可获得认证奖励的节点总量为1,443,497,378个，前14,436,417个认证的公民轻节点，每个认证的节点奖励9999.00元；第14,436,417个之后再认证的公民轻节点，每个认证的节点奖励999.00元；   
* 达到公民轻节点发行总量后，再认证的节点不再获得奖励，认证发行奖励以先完成认证优先获得，每个公民身份识别码仅能获得一次认证奖励；  
* 公民轻节点发行能让更多的公民参与公民建国运动，早日推翻共产党。

# 4.决议发行/resolution-issuance-iss
## 4.1.功能需求
* 成立联邦公民储备委员会后，由联储会决议发行数字公民币，经联储会联合会议决议通过后，由国储会权威节点提案发起发行；  
* 决议发行执行模块仅负责发行执行，提案、投票由决议发行治理模块（governance/resolution-issuance-gov）负责，治理模块通过后，发行模块执行发行；
* 发行模块根据决议发行提案铸造新公民币，所铸造的新币由各省储会地址（pallet_address）持有；
